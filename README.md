# Smart Front Desk — AI Lead Automation

An intelligent inbound lead system for local businesses.

## What it does
- Captures leads from WhatsApp, Email, and Web Forms
- Scores leads HOT/WARM/COLD using AI
- Sends personalized responses automatically
- Notifies business owner via SMS
- Logs everything to Airtable CRM
- Sends daily digest at 8am

## Stack
- n8n (automation)
- Groq (AI)
- Twilio (WhatsApp + SMS)
- Resend (Email)
- Airtable (CRM)

## How to use
1. Import both JSON files into your n8n instance
2. Replace API keys with your own
3. Publish and go live
